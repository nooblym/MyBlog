## 选择器

### 上下文选择器

1. 父元素 (**空格 子元素**)+

   会选择所有指定父元素下的指定子元素，包含子孙元素

2. 父元素 (**> 子元素**)+

   会选择所有指定父元素下的直系指定子元素，不包含孙元素

3. **标签1 + 标签2** 紧邻同胞选择器

   选择在标签1后面的标签2，必须紧跟

4. **标签1 ~ 标签2** 紧邻同胞选择器

   选择在标签1后面的标签2，不必紧跟

5. 通用选择器 *****

   单独使用会选择所有标签，可以配合其他选择器类型使用
